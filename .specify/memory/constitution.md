# Constitucion del producto

## Definition of Done

- La funcionalidad debe tener pruebas automatizadas.
- La cobertura de lineas debe ser igual o mayor a 80%.
- Semgrep no debe reportar vulnerabilidades criticas ni secretos expuestos.
- Cada Functional Requirement del `spec.md` debe estar cubierto por una prueba.
- Los casos de error y concurrencia no pueden quedar solo como validacion manual.
